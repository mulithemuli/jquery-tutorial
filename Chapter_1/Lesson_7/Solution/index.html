Thoughts on the solution

This solution creates a jQuery plugin which can be initialized by one line. We do not have the need to filter elements since the plugin takes care of it.

It has the advantage that the user just needs to call the plugin on the desired elements. It would be possible to pass an options object to the plugin which handles it or define methods which can be called on existing instances. But this will be covered later on.

The anonymous function defines its own namespace and does not pollute other namespaces (e.g. with the clickCounter variable). It also ensures that the $ variable is actually jQuery by passing jQuery in its self-call.

The rest of the functionality is the same as in lesson one.6
