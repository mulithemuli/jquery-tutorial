Thoughts on the solution

This solution adds a listener to the input text which triggers the click of the button which has been hidden.

In that approach we use the .trigger() function of jQuery. The change does not require a separate click but is automated. Although the event listener needs to be attached at separate points. We use the context aware selector to find the corresponding button.

The advantages to the previous approaches is that we do not need to click the button anymore and the update is done automatically. Since the button will be hidden by javascript and hidden afterwards it still will work when something in the selection goes wrong.
