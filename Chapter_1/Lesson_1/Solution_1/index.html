Thoughts on the solution

This solution uses hardcoded IDs which works but is not very generic. What if a third input should be added which covers the same functionality as one and two?

  • What have we done to make it work?

  • Added an id to the button
  • Added an id to the helper-text span
  • Attaching the event handler to each of the buttons separately
  • Fetching the value of the input in each of the handler functions separately
  • Writing the value to the span.helper-text by each id
